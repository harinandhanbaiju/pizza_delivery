🍕 Project Overview  
The application allows users to create their own custom pizza by selecting different ingredients and placing orders through an integrated online payment system. It also includes an admin dashboard to manage inventory, monitor orders, and track stock levels in real time.

🔹 Project Features  
• User and Admin login system  
• Complete registration, authorization, email verification, and forgot password functionality  
• Interactive dashboard displaying available pizza varieties  
• Custom pizza builder with multiple options for:  
 – Pizza base  
 – Sauce  
 – Cheese  
 – Veggies and other toppings  
• Razorpay Checkout Integration (Test Mode) for secure payment simulation  
• Automatic order confirmation after successful payment  

👨‍💻 Admin Panel Features  
• Mini inventory management system for pizza ingredients (base, sauce, cheese, veggies, meat)  
• Automatic stock updates after every order  
• Order status management (Order Received → In the Kitchen → Sent for Delivery)  
• Real-time status updates reflected in the user dashboard  
• Email notification system when stock falls below a threshold (e.g., pizza base < 20)  

🛠 Technologies Used  
• React.js  
• Node.js  
• Express.js  
• MongoDB  
• Razorpay API  
• HTML & CSS
